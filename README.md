# nhncloud-sdk-go

nhncloud-sdk-go is NHN Cloud Service SDK for Go.

## How to import

Reference the nhncloud-sdk-go package in your code :

```Go
import "github.com/cloud-barista/nhncloud-sdk-go"
```

Then update your `go.mod`:

```shell
go mod tidy
```
